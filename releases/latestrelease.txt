Dymola 2025x Refresh 1
2025-04-18

We are delighted to announce that Dymola 2025x Refresh 1 was released on April 18, 2025.

Simulation
- Simplified parameter optimization user interface
- Enhanced Monte-Carlo simulation
- Improved generation of analytic Jacobian

Model development
- Further improved unit checking
- No code generation that does not affect outputs (optional)
- More accurate parallelization for external functions

Environment
- Download and install libraries from GitHub
- Meta data to support creating the Commands menu in Dymola
